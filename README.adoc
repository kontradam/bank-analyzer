= Bank CSV Analyzer

Aplikacja konsolowa i graficzna (JavaFX) do analizy wyciągów bankowych w formacie CSV z automatyczną kategoryzacją transakcji i generowaniem raportów statystycznych.

Powstała z osobistej potrzeby do kontrolowania finansów na różnych kontach bankowych.

*Projekt na zaliczenie:* Laboratorium Specjalistyczne 2 +
*Autor:* kontradam +
*Rok akademicki:* 2025/2026

---

== Problem biznesowy

Ręczna analiza wyciągów bankowych jest czasochłonna i podatna na błędy. Aplikacja automatyzuje proces segregowania wydatków i generowania raportów finansowych, umożliwiając szybką analizę nawyków wydatkowych.

Automatyczna kategoryzacja transakcji na podstawie słów kluczowych w tytułach płatności oraz nazw kontrahentów.

Do aplikacji dołączone dwa pliki CSV z fake'owymi danymi do testowania.

== Funkcjonalności

* Multi-file processing - jednoczesne wczytywanie wielu plików CSV
* Automatyczna kategoryzacja - inteligentne przypisywanie transakcji do 10 kategorii
* Analiza wydatków - sumy i rankingi według kategorii
* Zestawienie miesięczne - przychody, wydatki i bilans dla każdego miesiąca
* TOP 10 wydatków - ranking najdroższych transakcji
* Export raportu - zapis pełnej analizy do pliku TXT
* Interfejs graficzny - wykresy wydatków i tabela transakcji (JavaFX)

== Instalacja i uruchomienie

=== Wymagania

Java 17 lub nowsza (JDK, nie JRE)

Sprawdzenie wersji:
----
java -version
----

=== Instalacja

Sklonuj repozytorium:
----
git clone https://github.com/kontradam/bank-analyzer.git
cd bank-analyzer
----

Zbuduj projekt:
----
gradlew.bat build
----

=== Uruchomienie

Interfejs graficzny (GUI):
----
gradlew.bat run
----

Aplikacja konsolowa:
----
java -jar build/libs/bank-analyzer-1.0.0.jar plik1.csv plik2.csv
----

=== Testy

----
gradlew.bat test
----

== Format pliku CSV

Aplikacja obsługuje format eksportu z PKO BP.

Format innych banków (mBank, ING, Santander) wymaga dostosowania parsera w klasie `CsvParserService`.
